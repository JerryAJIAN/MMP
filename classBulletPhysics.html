Rigid Body Creation

Creates a Rigid Body Object.

ã€€ When mass is 0 and kinematic is false, the object created becomes a static rigid body.
When mass is 0 and kinematic is true, the object can be moved manually, but it becomes a kinematic rigid body that isn't effected by physics calculation.

Parameters

heightThe object's height (y-length).
depthThe object's depth (z-length).

worldA pointer to the object's world transformation matrix. The center axis is the Z-axis.
massThe object's mass.
restitutionThe tendency of the object to return to its former shape after elastic deformation.
frictionThe amount of friction this object produces.
